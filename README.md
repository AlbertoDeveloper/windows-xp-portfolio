# windows-xp-portfolio

A Windows XP-inspired personal portfolio built with React and Vite.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## What to customize

Edit the file below first:

- `src/data/portfolio.js`

That file contains:

- name
- title
- summary
- email
- phone
- LinkedIn
- GitHub
- desktop shortcuts
- experience
- projects
- skills

## Structure

- `src/App.jsx`
- `src/components/DesktopIcon.jsx`
- `src/components/Window.jsx`
- `src/components/StartMenu.jsx`
- `src/components/Taskbar.jsx`
- `src/data/portfolio.js`
- `src/styles.css`
